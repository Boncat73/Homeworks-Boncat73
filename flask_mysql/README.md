# Система управління студентами (Flask + MySQL)

Це навчальний проєкт на Flask, який дозволяє переглядати та додавати записи про студентів у базу даних MySQL.

## Структура проєкту
- `app.py` — головний файл додатка.
- `templates/index.html` — візуальний інтерфейс сторінки.
- `requirements.txt` — список необхідних бібліотек.

## Вимоги
- Python 3.x
- MySQL Server

## Налаштування
