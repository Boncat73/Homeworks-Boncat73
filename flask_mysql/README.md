# Student Management System (Flask + MySQL)

Це повноцінний CRUD-додаток (Create, Read, Delete) для керування списком студентів. Проєкт демонструє роботу з Python, Flask та взаємодію з реляційною базою даних MySQL через SQLAlchemy.

## Функціонал
- **Перегляд списку:** Відображення всіх студентів із бази даних у реальному часі.
- **Додавання записів:** Форма для внесення імені та оцінки студента.
- **Видалення:** Можливість видаляти записи одним кліком із підтвердженням дії.
- **Стильний інтерфейс:** Адаптивний дизайн із використанням зовнішнього CSS.

## Структура проєкту
```text
├── static/
│   └── css/
│       └── style.css       # Зовнішні стилі додатка
├── templates/
│   └── index.html         # Шаблон головної сторінки (Jinja2)
├── app.py                 # Логіка Flask, моделі БД та маршрути
└── requirements.txt       # Залежності проєкту

## Встановлення
pip install -r requirements.txt
