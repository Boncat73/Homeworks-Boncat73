Технічний опис проєкту "Find Pets" (Портативна версія):

1. Огляд проєкту
Це веб-додаток на базі фреймворку Flask, призначений для публікації оголошень про 
зниклих або знайдених тварин. Проєкт розроблений як повністю портативний (Portable), 
працює з флеш-носія у зв'язці з середовищем MAMP.

2. Технологічний стек
Мова програмування: Python 3.13+

Backend: Flask (фреймворк)

Database: MySQL (через MAMP)

ORM: SQLAlchemy (для роботи з БД через об'єкти Python)

Frontend: HTML5, CSS3 (адаптивний дизайн)

Драйвер БД: PyMySQL

3. Структура директорій
Plaintext

/find_pets/
├── app.py              # Основна логіка сервера та маршрутизація
├── run_flask.command   # Скрипт для швидкого запуску на Mac
├── static/             # Статичні файли
│   ├── style.css       # Стилі інтерфейсу
│   └── uploads/        # Папка для завантажених фото тварин
├── templates/          # HTML-шаблони (Jinja2)
│   ├── index.html      # Головна сторінка
│   ├── login.html      # Сторінка авторизації
│   ├── register.html   # Сторінка реєстрації
│   └── add_pet.html    # Форма додавання оголошення
└── venv/               # Віртуальне середовище Python з бібліотеками
4. Налаштування Бази Даних
Проєкт використовує базу даних MySQL, що надається MAMP.

Назва БД: find_pets

Користувач/Пароль: root / root

Хост/Порт: localhost : 8889

Таблиці: * user: зберігає дані користувачів (id, username, password_hash, is_admin).

pet: зберігає оголошення (id, title, description, contact_info, image_file, user_id).

5. Інструкція із запуску (на новому пристрої)
Запуск БД: Відкрити програму MAMP на флешці та натиснути Start Servers.

Активація середовища: Відкрити Термінал у папці проєкту.

Запуск додатку: ```bash source venv/bin/activate python app.py

Доступ: Відкрити браузер за адресою http://127.0.0.1:5000.

6. Функціональні можливості
Система реєстрації та входу: Хешування паролів за допомогою werkzeug.security.

CRUD операції: Створення, читання та видалення оголошень.

Завантаження медіа: Обробка та збереження зображень у папку 
static/uploads з безпечною зміною імен файлів.

Рольова модель: Звичайні користувачі можуть створювати оголошення, 
адміністратор (is_admin=1) може видаляти будь-які записи.

7. Примітки для розробника
Для зміни прав адміністратора необхідно вручну змінити значення is_admin 
на 1 у таблиці user через phpMyAdmin.

При видаленні оголошення додаток автоматично видаляє пов'язаний файл 
зображення з файлової системи флешки для економії місця.
